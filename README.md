# Inventory Frontend (React + Vite)

## Features vs Criteria

- Authentication UI (login with JWT)
- Product CRUD (create, edit, delete, upload image)
- Order & stock management UI (create orders, view orders)
- Dashboard page (stats from /api/dashboard/stats)
- Responsive layout (sidebar collapses on small screens)
- Clean structure: api, context, pages, components, styles
- Ready for deployment (build with `npm run build`)

## Run

```bash
npm install
npm run dev
```

Make sure the backend is running at `http://localhost:4000`.
Make sure the backend is on seeve at `https://invetory-backend-2qkn.onrender.com/`.
The serve of frontend at `https://invetory-frontend-alpha.vercel.app/`
